데일리 손코딩 6일차
문제

정수 배열 nums와 정수 k가 주어진다.

길이가 k인 연속 부분 배열 중 합이 target 이상이 되는 부분 배열이 처음으로 등장하는 경우,
그 부분 배열의 시작 인덱스를 반환하라.

만약 조건을 만족하는 부분 배열이 없다면 -1을 반환한다.

입력 예시
nums = [2, 3, 1, 2, 4, 3]
k = 3
target = 7

출력 예시
1

제한 조건

1 ≤ k ≤ nums.length

nums.length ≥ 1

nums[i]는 0 이상의 정수

메서드 시그니처 (Java)
public int findFirstIndex(int[] nums, int k, int target)
