문제 2. 유효한 괄호 (Valid Parentheses)

문자열 s가 주어졌을 때,
괄호가 올바르게 짝지어져 있으면 true,
그렇지 않으면 false를 반환하는 함수를 작성하시오.

조건

열린 괄호는 같은 종류의 닫힌 괄호로 닫혀야 한다.

괄호는 올바른 순서로 닫혀야 한다.

문자열에는 다음 문자만 포함된다:
(, ), {, }, [, ]

입력 예시 1
s = "()"

출력 예시 1
true

입력 예시 2
s = "()[]{}"

출력 예시 2
true

입력 예시 3
s = "(]"

출력 예시 3
false

입력 예시 4
s = "([)]"

출력 예시 4
false

제약 사항

1 ≤ s.length ≤ 100,000
