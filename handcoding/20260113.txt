문제 1. 배열에서 두 수의 합

정수 배열 nums와 정수 target이 주어진다.
서로 다른 인덱스에 있는 두 원소의 합이 target이 되는
인덱스 쌍을 반환하는 함수를 작성하시오.

조건

정확히 하나의 정답만 존재한다.

같은 원소를 두 번 사용할 수 없다.

반환 순서는 상관없다.


입력 예시 1
nums = [2, 7, 11, 15]
target = 9

출력 예시 1
[0, 1]

입력 예시 2
nums = [3, 2, 4]
target = 6

출력 예시 2
[1, 2]

제약 사항

2 ≤ nums.length ≤ 100,000

-1,000,000,000 ≤ nums[i] ≤ 1,000,000,000

-1,000,000,000 ≤ target ≤ 1,000,000,000
