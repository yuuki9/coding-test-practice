문제: 최소 길이 부분 배열
문제 설명

정수 배열 nums와 정수 target이 주어진다.
합이 target 이상이 되는 연속 부분 배열 중
길이가 가장 짧은 것의 길이를 구하시오.

조건을 만족하는 부분 배열이 없다면 0을 반환하시오.

입력 예시
target = 7
nums = [2,3,1,2,4,3]

출력 예시
2


설명: [4,3]의 합이 7이고, 길이가 2로 가장 짧다.

조건

1 ≤ target ≤ 10⁹

1 ≤ nums.length ≤ 10⁵

1 ≤ nums[i] ≤ 10⁴

제약 (중요)

❌ 모든 부분 배열을 다 계산하면 시간 초과

⭕ O(n) 으로 풀 수 있어야 함

❗ 연속 부분 배열임
